JavaScript

Introduction

Javascript is a programming language that makes web applications more dynamic and interactive.It is used to create or build the functionality to the website.Javascript had multiple definitions like

  • Javascript is a lightweight,cross-platform,single-threaded and interpreted compiled programming language.
  • Javascript is a scripting language and used to provide functionality to webpages.
  • Javascript is compiled programming language.
  • Javascript is a single-threaded language.
  • Javascript is a synchronous single-threaded language.
  • Javascript is a high-level language.

Synchronous single-threaded:

JavaScript is primarily synchronous,JavaScript operates on a single thread within the browser's runtime environment. This means that it can only perform one task line by line in the order it appears in the script at a time from start to finish.

High-level:

JavaScript is designed to be easy to read and write, abstracting away low-level details and providing developers with a more intuitive syntax.

Interpreted:

JavaScript code is executed line by line by an interpreter, rather than being compiled into machine code before execution. This allows for rapid development and easy debugging but may result in slightly slower performance compared to compiled languages.

Scripting Language:

JavaScript is often used for scripting tasks, especially within web development. It's commonly embedded within HTML pages and executed by web browsers to provide dynamic and interactive functionality.

Document Object Model (DOM)/ Browser Object Model (BOM):

JavaScript interacts with the DOM, which represents the structure of HTML documents as a tree of objects. This allows JavaScript to manipulate the content, structure, and style of web pages dynamically, enabling interactivity and responsiveness. Browser Object Model (BOM) in JavaScript provides access to browser-related features like controlling windows, manipulating the document (web page), navigating browser history, and retrieving information about the user's screen and browser environment. It enables JavaScript to interact with the browser itself, beyond just the content of web pages.
